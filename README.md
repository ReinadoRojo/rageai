# RageAI
---

I made this funny and basic chat app using HackAI to get some laught while an AI model (gemini-2.5-flash) ragebaits me.

## Tools

Here are the tools I used

- NextJS
- shadcn-ui
- Vercel (for hosting)
- VSCode
- Hackatime (time tracking)
- HackAI (AI provider)
- Bun
